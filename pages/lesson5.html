GIT flow

GIT clone - для того, чтобы "склонить"репозиторий
GIT pull - для того, чтобы получить именения с репозитория
GIT checkout -b branchname - создать новую ветку и переключится на нее
GIT add . - добавить именения, которые сделали локально
GIT commit -m "coment text" - подписать их
GIT push origin branchname - запушить на git сервер
